# Duplicate References

**Duplicate references** makes an exact copy of one or more existing entries, spliced into the source right after each original — a starting point for cross-posting the same material under a different heading, without retyping it from scratch.

## How to use it

1. Select one or more rows in the [entry table](../entry_table/editing.md).
2. Right-click and choose **Duplicate references**.
3. Each selected entry is copied — same page style, same location — with a fresh copy inserted immediately after the original in the source.
4. The new copies start out identical to the originals. Retype the Main/Sub1/Sub2 heading on each new row to point it at the different heading you actually want it cross-posted under.

## Range references

If a selected entry is a [range reference](../index_tree/range_references.md), duplicating it copies **both** halves — the opener and the closer — as a new, correctly linked pair, not just the visible opener. You end up with a second, independent range rather than an orphaned opener or closer.
